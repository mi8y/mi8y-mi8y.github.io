Countries++ (GraphQL) API Documentation

Countries++ GraphQL API provides a comprehensive GraphQL API to query information on Countries, States/Provinces, Cities, Languages, Currencies, Timezones, Continents and World Regions. Here you'll find detailed documentation of various types and queries possible and discover various ways to use this API to the maximum capability.

Connect and Access this API for FREE at Rapid API

Entities

#

1. Continents

#

  • Alpha-2 Continent Code
  • Name of the Continent (eg. North America )
  • Countries in the Continent -> Countries

2. Regions

#

  • Name of the Region (eg. Americas )
  • Subregions in this Region -> Subregions

3. Subregions

#

  • Name of the Subregion (eg. North America )
  • Countries in this Subregion -> Countries

4. Countries

#

  • Codes
    • ISO 3166-1 Alpha-2 Country Code (eg. US )
    • ISO 3166-1 Alpha-3 Country Code (eg. USA )
    • ISO 3166-1 Numeric-3 Country Code (eg. 840 )
    • International Olympics Commitee (IOC) Country Code (eg. USA )
    • FIFA Country Code (eg. USA )
  • Names in various languages
    • Common Country Name (eg. United States )
    • Official Country Name (eg. United States of America )
    • Language -> Languages
  • Flag

  • Sovereignity
    • Independent Status (eg. true
    • Assignment Status (eg. officially-assigned )
  • Geographic information
    • Continent, this Country belongs to -> Continent
    • Region, this Country belongs to -> Continent
    • Subregion, this Country belongs to -> Continent
    • Bordering Countries -> Countries

  • Currencies in use -> Currencies

  • Country Code Top-level Domains (ccTLD) (eg. .us )
  • International Direct Dialing (ISD/IDD codes) (eg. +1201, +1202 ... +1986, +1989 )
  • Demonyms
    • Male (eg. American )
    • Female (eg. American )
    • Language -> Languages

5. Subdivisions

#

  • ISO 3166-2 Subdivision Code (eg. US-CA )
  • Name of the Subdivision (eg. California )
  • Geo-Coordinates of the Subdivision (eg. latitude: 36, longitude: -119 -> 36Â°N, 119Â°E )
  • Cities in this Subdivision -> Cities
  • Country this Subdivision belongs to -> Countries

6. Cities

#

  • Name of the City (eg. Palo Alto )
  • Geo Coordinates of the City (eg. latitude: 37, longitude: -122 -> 37Â°N, 122Â°E )
  • Subdivision, this City belongs to -> Subdivisions
  • Country, this City belongs to -> Countries

7. Languages

#

  • ISO 693-1 Alpha-2 Language Code (eg. EN )
  • ISO 693-3 Alpha-3 Language Code or ISO 693-5 Alpha-3 Macrolanguage Code (eg. ENG )
  • Name of the Language (eg. English )
  • Countries where this Language is spoken -> Countries

8. Currencies

#

  • ISO 4217 Alpha-3 Currency Code (eg. USD )
  • Name of the Currency (eg. United States dollar )
  • Symbol of the Currency (eq. $ )
  • Countries using this Currency -> Countries

9. Timezones

#

  • Timezone Abbreviation (eg. PST )
  • Name of the Timezone (eg. Pacific Standard Time (North America )
  • Zone Name (eg. America/Los_Angeles )
  • Timezone UTC Offset in Seconds (eg. -28800 )
  • Timezone UTC Offset ID (eg. UTC-08:00 )
  • Countries where this timezone is applicable -> Countries
