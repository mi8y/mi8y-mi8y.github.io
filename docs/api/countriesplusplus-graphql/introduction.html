Introduction

Welcome to Countries++ GraphQL API Documentation

A comprehensive GraphQL API to query information on Countries

  • Codes
    • ISO 3166-1 Alpha-2 Code (eg. US )
    • ISO 3166-1 Alpha-3 Code (eg. USA )
    • ISO 3166-1 Numeric-3 Code (eg. 840 )
    • International Olympics Commitee (IOC) Code (eg. USA )
    • FIFA Code (eg. USA )
  • Names in various languages
    • Common Name (eg. United States )
    • Official Name (eg. United States of America )
    • Language (eg. English )
  • Flag
    • Emoji (eg. 🇺🇸 )
    • SVG (eg. https://flagcdn.com/us.svg )
  • Capital (eg. Washington D.C. )
  • Subdivisions (i.e. States/Provinces etc.)
    • ISO 3166-2 Code (eg. US-CA )
    • Name (eg. California )
    • Geo-Coordinates (eg. latitude: 36, longitude: -119 -> 36°N, 119°E )
    • Cities
      • Name (eg. Palo Alto )
      • Geo Coordinates (eg. latitude: 37, longitude: -122 -> 37°N, 122°E )
  • Geographic information
    • Continent (eg. North America )
    • Region (eg. Americas )
    • Subregion (eg. North America )
    • Bordering Countries (eg. Canada, Mexico and dive further)
    • Landlocked (eg. false i.e. have bordering seas)
    • Area in sq.km (eg. 9372610 )
    • Coordinates (eg. latitude: 38, longitude: -97 -> 38°N, 97°E )
  • Sovereignity
    • Independent (eg. true
    • Assignment Status (eg. officially-assigned )
  • Languages
    • ISO 693-1 Alpha-2 Code (eg. EN )
    • ISO 693-3 Alpha-3 Code or ISO 693-5 Alpha-3 Code for Macrolanguages (eg. ENG )
    • Name (eg. English )
  • Currencies
    • ISO 4217 Alpha-3 Code (eg. USD )
    • Name (eg. United States dollar )
    • Symbol (eq. $ )
  • Timezones
    • Abbreviation (eg. PST )
    • Name (eg. Pacific Standard Time (North America )
    • Zone Name (eg. America/Los_Angeles )
    • UTC Offset in Seconds (eg. -28800 )
    • UTC Offset ID (eg. UTC-08:00 )
  • Country Code Top-level Domains (ccTLD) (eg. .us )
  • International Direct Dialing (ISD/IDD codes) (eg. +1201, +1202 ... +1986, +1989 )
  • Demonyms
    • Male
    • Female
    • Language
